An awesome vibe coded Shelly + Ruuvi tag thing to keep my crawl space dry!

This uses a Shelly plug and two Ruuvi tags to compare humidity levels and turn on a fan based on which air is more dry.
The point is to run everything locally on one shelly device so no wifi needed.

Equipment needed:
1 x PlugS Gen3
2 x regular Ruuvi tag with temperature and humidity sensors

Sorry not sorry the comments are in ancient elf language.
